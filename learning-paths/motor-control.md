## 1. 路线定位

这条路线适合准备进入执行机构、控制回路和机器人驱动方向的读者。

- 适合人群：已经接触过单片机，想进一步做电机、驱动和反馈控制的人
- 学完收获：能理解 PWM、驱动器、编码器和速度环/位置环的基本工作方式
- 适合延伸：机器人执行器、闭环控制、运动平台调试

## 2. 先修要求

- 数学：建议掌握比例、微分变化和最基础的函数概念
- 编程：建议已经会写基础单片机程序和中断/定时器逻辑
- 硬件：理解 H 桥、驱动器、电源、编码器和传感器基础会更轻松

## 3. 推荐环境

- 板卡与器件：一块 MCU 开发板、一个驱动器、一台小电机和编码器
- 工具链：串口日志工具、示波器或逻辑分析仪、基础调参记录工具
- 软件环境：VS Code 或厂商 IDE、数据记录脚本、简单绘图工具

## 4. 四阶段路线图

### 入门基础 / Foundation

- 学习目标：先看懂驱动链路和执行机构的最小控制方法
- 核心知识点：PWM、方向控制、电源、驱动器接口、安全停机
- 对应内容：后续会补电机控制最小硬件链路和基础驱动

### 核心原理 / Core

- 学习目标：理解反馈、测速、位置估算和闭环控制基础
- 核心知识点：编码器、采样周期、速度估算、位置环、控制回路
- 对应内容：后续会补反馈获取、状态计算和参数理解

### 实战项目 / Practice

- 学习目标：把驱动、反馈和调参整合成一个可重复运行的小系统
- 核心知识点：日志、状态观测、参数记录、故障保护、实验验证
- 对应内容：后续会补闭环调试和最小控制系统实践

### 进阶扩展 / Advanced

- 学习目标：开始把控制系统放到机器人或复杂执行平台里思考
- 核心知识点：多环控制、执行器接口、系统延迟、整机协同
- 对应内容：后续会补机器人平台中的执行机构控制
